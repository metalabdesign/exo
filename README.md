# Exo

A Thorax/Backbone UI library.

## Installation

Add this line to your application's Gemfile:

```ruby
    gem 'exo'
```

And then execute:

```sh
$ bundle
```

And then, in your asset manifest files, require as much or as little as you'd like:

```coffee
#= require exo
#
# OR
#
#= require exo/views
#= require exo/mixins
#= require exo/widgets
```
